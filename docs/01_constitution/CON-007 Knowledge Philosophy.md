# 📘 CON-000
# 제7장 지식 철학 (Knowledge Philosophy)

---

## 제35조 (지식의 정의)

My Development Ecosystem(MDE)에서 **지식(Knowledge)** 이란 단순한 문서나 코드가 아니다.

지식은 프로젝트 수행 과정에서 생성되는 모든 정보와 경험을 의미하며, 미래 프로젝트에서 재사용 가능한 자산이다.

MDE는 프로젝트보다 지식을 오래 유지하는 것을 목표로 한다.

---

## 제36조 (Knowledge First 원칙)

모든 프로젝트는 종료와 동시에 하나 이상의 지식 자산을 남겨야 한다.

프로젝트가 종료되어도 지식은 계속 활용될 수 있어야 한다.

프로젝트의 성공은 프로그램 완성뿐 아니라 지식 자산의 축적으로도 평가한다.

---

## 제37조 (지식 자산의 분류)

MDE는 지식을 다음과 같이 분류한다.

### 1. 설계 지식

- 요구사항
- SDS
- 시스템 아키텍처
- DB 설계
- API 설계
- UI 설계

---

### 2. 구현 지식

- 소스 코드
- 코드 템플릿
- 라이브러리
- 컴포넌트
- 플러그인

---

### 3. 운영 지식

- 오류 해결 사례
- 성능 개선 기록
- 운영 절차
- 배포 기록

---

### 4. AI 지식

- 프롬프트
- 프롬프트 템플릿
- Agent 역할
- Workflow
- AI 결과 평가

---

### 5. 학습 지식

- 회고
- 기술 조사
- 실험 결과
- 아이디어
- 참고 자료

---

## 제38조 (지식 생명주기)

모든 지식은 다음 생명주기를 따른다.

```text
생성

↓

검토

↓

분류

↓

저장

↓

검색

↓

재사용

↓

개선

↓

버전 관리

↓

영구 보관
```

지식은 일회성 자료가 아니라 지속적으로 개선되는 자산이다.

---

## 제39조 (Knowledge Platform)

MDE는 하나의 중앙 Knowledge Platform을 운영한다.

Knowledge Platform은 다음 기능을 제공한다.

- 문서 저장
- 코드 저장
- 프롬프트 저장
- 오류 사례 저장
- 템플릿 저장
- 검색
- 버전 관리
- 프로젝트 간 연결

Knowledge Platform은 프로젝트와 독립적으로 운영되며, 모든 프로젝트가 공통으로 활용한다.

---

## 제40조 (지식의 연결성)

모든 지식은 서로 연결 가능해야 한다.

예를 들어,

- 요구사항 ↔ SDS
- SDS ↔ 코드
- 코드 ↔ 테스트
- 테스트 ↔ 오류
- 오류 ↔ 해결 방법
- 해결 방법 ↔ 템플릿
- 템플릿 ↔ 다음 프로젝트

이러한 연결을 통해 하나의 지식 그래프를 형성한다.

---

## 제41조 (재사용 원칙)

새로운 프로젝트를 시작할 때는 먼저 기존 지식을 검색한다.

다음 순서를 따른다.

1. 기존 프로젝트 검색
2. 기존 템플릿 검색
3. 기존 코드 검색
4. 기존 프롬프트 검색
5. 기존 오류 해결 사례 검색
6. 기존 설계 문서 검색

재사용 가능한 자산이 있다면 우선 활용하고, 필요 시 개선하여 사용한다.

---

## 제42조 (AI와 지식)

AI는 Knowledge Platform을 활용하여 프로젝트를 지원한다.

AI는 다음 활동을 수행할 수 있다.

- 관련 문서 검색
- 유사 프로젝트 탐색
- 템플릿 추천
- 프롬프트 추천
- 코드 재사용 제안
- 오류 해결 사례 제안

AI는 저장된 지식을 활용하여 일관성과 생산성을 향상시킨다.

---

## 제43조 (지식 품질)

모든 지식은 다음 기준으로 관리한다.

□ 최신 상태 유지

□ 출처 기록

□ 변경 이력 관리

□ 검색 가능

□ 재사용 가능

□ 프로젝트 연결

□ 표준 형식 준수

---

## 제44조 (지식의 궁극적 목표)

MDE의 궁극적인 목표는 다음과 같다.

> "프로젝트를 만드는 것이 아니라, 시간이 지날수록 성장하는 개인의 지식 운영체계를 구축한다."

모든 프로젝트는 지식을 만들고,

모든 지식은 다음 프로젝트를 더욱 빠르고 정확하게 수행할 수 있도록 지원한다.

---

## 제7장 자체 검토

✓ 지식의 정의 완료

✓ 지식 자산 분류 완료

✓ 지식 생명주기 정의 완료

✓ Knowledge Platform 정의 완료

✓ AI와의 연계 정의 완료

✓ 재사용 원칙 정의 완료
